做构建前端项目时 还需要进行一下操作优化所以还需要依赖以下gulp插件来实现。

  1. 图片（压缩图片支持jpg、png、gif）
  2. 样式 （支持sass 同时支持合并、压缩、重命名）
  3. javascript （检查、合并、压缩、重命名）
  4. html （压缩）
  5. 客户端同步刷新显示修改
  6. 构建项目前清除发布环境下的文件（保持发布环境的清洁）所需插件如下］

gulp plugins

  1. 通过gulp plugins，寻找对于的gulp组件
  2. gulp-imagemin: 压缩图片
  3. gulp-sass: 支持sass
  4. gulp-csso: 压缩css
  5. gulp-jshint: 检查js
  6. gulp-uglify: 压缩js
  7. gulp-concat: 合并文件
  8. gulp-rename: 重命名文件
  9. gulp-htmlmin: 压缩html
  10. gulp-clean: 清空文件夹
  11. gulp-livereload: 服务器控制客户端同步刷新（需配合chrome插件LiveReload及tiny-lr）

项目目录大致为如下

  1. project(项目名称)
  2. |___node_modules (组件目录
  3. |___dist 发布环境
  4. |__css 样式文件(style.css style.min.css)
  5. |__images 图片文件(压缩图片)
  6. |__js js文件(main.js main.min.js)
  7. |__index.html 静态文件(压缩html)
  8. |___src 生产环境
  9. |__sass scss文件
  10. |__images 图片文件
  11. |__js javascript文件
  12. |__index.html 静态文件
  13. |__gulpfile.js gulp任务文件

gulp基础语法

  1. gulp通过gulpfile文件来完成相关任务，因此项目中必须包含gulpfile.js

  2. gulp有五个方法：src、dest、task、run、watch

  3. src和dest：指定源文件和处理后文件的路径

  4. watch：用来监听文件的变化

  5. task：指定任务

  6. run：执行任务


/*!
 * gulp
 * $ npm install gulp-sass gulp-minify-css gulp-jshint gulp-concat gulp-uglify gulp-imagemin  gulp-rename gulp-livereload gulp-cache del --save-dev
 */

// Load plugins
var gulp = require('gulp'),
    sass = require('gulp-ruby-sass'),
    minifycss = require('gulp-minify-css'),
    jshint = require('gulp-jshint'),
    uglify = require('gulp-uglify'),
    imagemin = require('gulp-imagemin'),
    rename = require('gulp-rename'),
    concat = require('gulp-concat'),
    notify = require('gulp-notify'),
    cache = require('gulp-cache'),
    livereload = require('gulp-livereload'),
    del = require('del');

// Styles
gulp.task('styles', function() {
  return gulp.src('src/styles/main.scss')
    .pipe(sass({ style: 'expanded', }))
    .pipe(gulp.dest('dist/styles'))
    .pipe(rename({ suffix: '.min' }))
    .pipe(minifycss())
    .pipe(gulp.dest('dist/styles'))
});

// Scripts
gulp.task('scripts', function() {
  return gulp.src('src/scripts/**/*.js')
    .pipe(jshint('.jshintrc'))
    .pipe(jshint.reporter('default'))
    .pipe(concat('main.js'))
    .pipe(gulp.dest('dist/scripts'))
    .pipe(rename({ suffix: '.min' }))
    .pipe(uglify())
    .pipe(gulp.dest('dist/scripts'))
});
// Images
gulp.task('images', function() {
  return gulp.src('src/images/**/*')
    .pipe(cache(imagemin({ optimizationLevel: 3, progressive: true, interlaced: true })))
    .pipe(gulp.dest('dist/images'))
});
// Clean
gulp.task('clean', function(cb) {
    del(['dist/assets/css', 'dist/assets/js', 'dist/assets/img'], cb)
});
// Default task
gulp.task('default', ['clean'], function() {
    gulp.start('styles', 'scripts', 'images');
});

// Watch
gulp.task('watch', function() {
  // Watch .scss files
  gulp.watch('src/styles/**/*.scss', ['styles']);
  // Watch .js files
  gulp.watch('src/scripts/**/*.js', ['scripts']);
  // Watch image files
  gulp.watch('src/images/**/*', ['images']);
  // Create LiveReload server
  livereload.listen();
  // Watch any files in dist/, reload on change
  gulp.watch(['dist/**']).on('change', livereload.changed);
});
